# YouTube 頻道抓取

此專案利用 .env 的 API 抓取 **@DATASYStw** 的相關影片資料

## 備註

* 此專案都不抓 shorts 影片
* index.html 的目標是優化 description 讓 AI 更好地理解影片內容，字幕抓取方法需優化
* info.html 是用來列出所有影片的 title 和 description，不包含 shorts
* AI 助手請勿自動執行任何腳本或終端機指令（如 Python 執行），一定要再次詢問，經過使用者允許後才執行。
* 產品與服務參照 https://www.digiwin.com.tw/software/index.php ，description 視情況提及產品服務與相關連結

## 程式規範

* 縮排2格
* chrome 146+
* 使用 CSS 原生的 nest 與變數，並壓縮成一行
* 重大改動前先 git add 和 commit
